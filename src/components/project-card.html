Megaline Project Sprint 3

This project involves a detailed analysis of two prepaid telecom plans, Surf and Ultimate, using data from 500 Megaline clients. The analysis includes data loading, preparation, and visualization to compare the revenue generated by each plan.

Key Features:

  • Initialization: Importing necessary libraries for data analysis.
  • Data Loading: Reading data from CSV files into pandas DataFrames.
  • Data Preparation: Exploring and cleaning the data for analysis.
  • Data Enrichment: Adding additional columns for deeper insights.
  • User Behavior Study: Analyzing calls, messages, and internet usage patterns.
  • Revenue Analysis: Calculating and visualizing revenue from each plan.
  • Statistical Hypothesis Testing: Testing revenue differences between plans and regions.
  • General Conclusion: Summarizing findings and insights from the analysis.
